# Theme Configuration
#
# This file configures which visual theme to use for the game.
# The selected theme's CSS will be loaded instead of the default.
#
# Format:
#   theme_name [ ]  - Available but not selected
#   theme_name [x]  - Selected (only one can be selected)
#
# Available themes (prototype first, then alphabetical):
#   prototype     - Original warm beige VN style (cream text box, golden buttons)
#   70s           - Retro disco (orange, brown, gold, avocado green)
#   80s           - Synthwave neon (hot pink, electric blue, chrome)
#   90s           - Windows 3.1 / Early Mac aesthetic (gray panels, blue titlebars)
#   cyberpunk     - Neon city (yellow/cyan on black, grid patterns, tech noir)
#   dos           - MS-DOS terminal (green on black, CRT scanlines, command prompt)
#   dragonball    - Dragon Ball Z inspired (orange/blue energy, power-up auras)
#   finalfantasy  - Classic JRPG menu (blue gradients, white text, crystal motifs)
#   gameboy       - Original Game Boy (4-shade green palette, pixel font)
#   harrypotter   - Wizarding World (burgundy/gold, magical sparkles, parchment)
#   lotr          - Lord of the Rings (parchment, gold, forest green, elvish elegance)
#   manga         - Junji Ito horror manga (stark black & white, high contrast)
#   nes           - 8-bit NES (limited palette, pixel font, chunky borders)
#   snes          - 16-bit SNES (richer colors, gradients, star field)
#   spaceopera    - Epic sci-fi (deep space, nebula colors, starfield, hologram cyan)
#   starwars      - Star Wars inspired (space black, yellow crawl text, sci-fi)
#   vaporwave     - A E S T H E T I C pink/cyan/purple with grid patterns
#   y2k           - Millennium aesthetic (chrome silver, translucent, tech optimism)
#   zelda         - Legend of Zelda (forest green, Triforce gold, adventure RPG)
